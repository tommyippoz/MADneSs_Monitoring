HOW TO RUN MADNESS

Download
- Download the executable .jar
- Download Input Files
- Download one or more archives containing preprocessed datasets

Setup
- create a new folder e.g., myfolder
- put the JAR in the folder
- uncompress input files in the folder ("input" folder should be in the same folder as the jar file)
- uncompress datasets in the folder ("datasets" folder should be in the same folder as the jar file)

Running
- from shell, run "java -jar myfolder/RELOAD_V3.jar"
